Send Connectors


Send connector allows Exchange to send mail through to the Internet.
  1. Once logged into the Web Console open the mail flow from the left panel.
  2. Open the Send Connector from the top panel
  3. Click the + sign
  4. add a name for send connect
  5. select "Internet"
  6. Click Next

For this example we are not going to smart host for send mail, we will just the MX record tied to the domain.
So for the next window you can leave it as is and just click next


Next you configure the routing of mail to specific domains, now for me I want this to send to all domains.  First step is to click the Plus sign,  After that add * to Full Qualified Domain Name field and click save.



The last step is to add the server to send mail from, this can be any transport server or edge server.  We haven't set up a edge server yet but will in another blog post.   So for now there is only the one server we can select and thats the one you are working on now.  First click the + sign, then add the server and click ok.  That brings you to the last screen where you can now see the server and you can click finish.




Now that is the send connector done.  From here you should now be able to send emails.
