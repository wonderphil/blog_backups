What Is Selenium-Cucumber?

First what is Selenium, well I'll steel this part straight from the source of truth: (http://www.seleniumhq.org/)

Selenium automates browsers. That's it! What you do with that power is entirely up to you. Primarily, it is for automating web applications for testing purposes, but is certainly not limited to just that. Boring web-based administration tasks can (and should!) also be automated as well.

Selenium has the support of some of the largest browser vendors who have taken (or are taking) steps to make Selenium a native part of their browser. It is also the core technology in countless other browser automation tools, APIs and frameworks.

Second Cucumber, because I am lazy I steel this as well: (https://en.wikipedia.org/wiki/Cucumber_(software))

Cucumber is a software tool that computer programmers use for testing other software. It runs automated acceptance tests written in a behavior-driven development (BDD) style. Cucumber is written in the Ruby programming language.Cucumber projects are available for other platforms beyond Ruby.


What are we using this for?

Like I said before I am lazy and I hate repeating myself, really hate repeating myself.  So we use this to do some automated Web testing and also set this up for mobile web testing. 

Prerequisites

Firstly let's get all the little bits of software installed, I am running this on a Windows 10 VM, basic 4gb ram, 50gb disk and 2 cores (in now way this the recommended just info on how its setup)

Must Haves:
  1. Download Ruby - http://rubyinstaller.org/downloads/
    I am using version 2.3.1 64bit
  2. Devkit which is also downloadable from the above link
  3. Selenium-Cucumber Gem

    The following is required for mobile testing 
  4. Java JDK - http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
    Required for Android
  5. Android SDK - https://developer.android.com/studio/install.html
    I just downloaded the studio which includes the SDK
  6. Appium - https://bitbucket.org/appium/appium.app/downloads/AppiumForWindows.zip
Optional:
  1. Some text editior, I use sublime - https://www.sublimetext.com/3
  2. Chrome, Firefox and any of your other browsers
  3. Chrome driver, you need this for testing in chrome -  http://chromedriver.storage.googleapis.com/index.html?path=2.9/
  4. Internet Explorer driver, again if you want to test in IE - http://selenium-release.storage.googleapis.com/index.html?path=2.42/
  5. Ansicon, this is a cool little tool to give you better readability of the output - https://github.com/adoxa/ansicon/downloads
  6. Git, as everything should be source controlled - https://git-scm.com/download/win

Installing everything in the same order as above:

I won't go through the whole install process but I would suggest the following:


DevKit, 
  1. Download is a self-extracting archive and extract,
  2. Open CMD with Ruby started to that location


  3. Add Ruby to config file
    c:\devkit\ruby dk.rb init

  4. Then install DevKit
    c:\devkit\ruby dk.rb install

Appium
Ok so Appium requires that you have installed NodeJs, I will show how to do that here:
  1. Using the Installer downloaded from the website - https://nodejs.org/en/download/ I have just used the default options
  2. You will also need to have .NET Framework v4.5.1 installed -  https://www.microsoft.com/en-gb/download/details.aspx?id=40773
  3. Now make sure that you have run and configured Android Studio before going ahead, also that you have a java_home path set
  4. Now we need to setup some Environment variables 
    set ANDROID_HOME=C:\<installation location>\android-sdk-windows

    set PATH=%PATH%;%ANDROID_HOME%\tools;%ANDROID_HOME%\platform-tools;%JAVA_HOME\bin
  5. Open up cmd again and type the following command
    npm install appium-doctor -g

  6. Now run the following command, this will tell you if Appium is ready
    appium-doctor

    You should see something like this and if you don't you need to fix any errors


  7. Now that is all done, download and run the Annium exe from the above link.

This should now complete everything you need to install and setup.
